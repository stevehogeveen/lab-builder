# Module: qnap

## Purpose
Implements QNAP configuration and validation actions for kits that include QNAP.

## Key Files
- `app/modules/qnap/manifest.yml`
- `app/modules/qnap/routes.py`
- `app/modules/qnap/service.py`
- `app/modules/qnap/schemas.py`

## Update Guide
- Change UI actions in `routes.py`.
- Update QNAP behavior in `service.py`.
- Update schema fields/validation in `schemas.py`.
- Keep manifest navigation aligned with section routing.

## Validation
Enable QNAP, save config, and verify preview/status actions.
