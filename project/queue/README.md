# Product Queue

This is the LabBuilder product-local queue for Product Team Beta work under Project OS.

Use this queue for product-owned implementation, testing, documentation, and release tasks.

Project-wide standards, policies, templates, bridge protocols, Sentinel behavior, and reusable operating practices should be recommended for promotion to Project OS instead of being redesigned here.

## Current Folders

- `ready/`: tasks that are clear enough to start.

## Queue Rules

- Keep tasks small enough to verify.
- Prefer reversible changes.
- Do not run destructive hardware workflows without explicit approval.
- Report cross-product patterns as promotion candidates for Project OS.
